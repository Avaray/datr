I created this package for my personal usage because I like to use this format of date.  
Maybe someone of you will find it useful.

Install:
> npm i datr

Usage example:
> const datr = require('datr')  
>
> console.log(datr())  
> // output: 20181230183500
>
> console.log(datr(1))  
> // output: 20181230.183500
>
> console.log(datr(2))  
> // output: 20181230
